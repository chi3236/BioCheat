Calculate Restrict Map From Electrophoresis Result

How to use

Input your electrophoresis result. Base length will be calculated if not given, using given base length. You can choose regression method using when calculate base length.

You can choose digest manner(double, partial) and DNA form(linear, circular). Restrict map will be calculated usgin electrophoresis data. Candidates are sorted by affinity.

This application does not estimate overlapped marker. If you want apply overlapped marker, enter repeatedly.(ex: A: 2.5 3 3)

Format

[tag]: [transition distance]-[known base length] [transition distance]-[known base length]

Example

ladder: 2.6-10000 2.8-8000 3.1-6000 3.3-5000 3.6-4000 4-3000 4.6-2000 5.1-1500 5.8-1000
A: 2.5 5.5 6.4
