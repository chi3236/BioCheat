Amino Acid Sequence to Codon Sequence Converter

It will convert your amino acid sequence to codon sequence according to organism's codon adaptation index you selected. You can select convert manner.

How to use

Search organism to compare. You need to input least 3 characters to search organism in our database (datasets are from ftp.kazusa.or.jp/pub/codon/current/). Select translation table. You can change values clicking show button.

Input your amino acid sequence. Character except alphabet will be ignored (not case sensitive). You can select 3 letter format (Ala for Alanine) or 1 letter format (A for Alanine). Or you can just upload your fasta file.

You can select convert manner. 'Consider Adation Index' will convert according to selected organism's Codon Ratio Table. 'Highest Adaption Index only' will choose highest adaption index if there's many candidates. 'Show All Probability' will show you all probability.

Example

If Alanine is translated by GCU and GCC. And each adaption indexes are 1 and 0.5 
Consider Adaption Index:
	Ala-Ala-Ala -> GCU-GCC-GCU
Hieghest Adaption Index only:
	Ala-Ala-Ala -> GCU-GCU-GCU

You can download resut as FASTA file.
