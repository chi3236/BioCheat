Base Skew Plotter

How to use

Select two bases to calculate skew.

Skew

skew = ([#base1 in window] - [#base2 in window]) / ([#base1 in window] + [#base2 in window]

if #base1 + #base2 is 0 : skew=0

You can change window size and window pivot. Pivot moves scanning window backwards. If pivot is 1, scanning window starts 1 base backwards.
